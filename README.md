# NUMBLES (C64)

Numbles is a small BASIC game for the Commodore 64 inspired by classic titles like Snake and Nibbles.

## Gameplay

Control a worm made of asterisks and collect numbers from 0 to 9.

Controls:
W = up  
S = down  
A = left  
D = right  

## Mechanics

- Numbers 1–9 increase score and grow the worm  
- 0 is a bonus: it shortens the worm  

## Run

```basic
LOAD"NUMBLES",8,1
RUN
```
